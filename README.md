# HealthMax
A virtual health companion to assist healthcare providers to receive health and biomarker data from registered patients. The application connects to native health companions in phones, watches, and other compatible devices to record selected biomarker data. This project is part of the F29SO course at HWUM, 2025-26.

## How to Run

1. Ensure you have flutter installed along with an emulator. We recommend a phone emulator for the best experience.
2. Go to `healthmax-frontend/` folder and run `flutter run`

## How to view deployed version

Visit it at `f29so-healthmax.netlify.app/`
